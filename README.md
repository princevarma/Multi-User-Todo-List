# Multi-user-todo-list
This is Multi-User Todo List Using Django in which User can register ,login themselves and add/view/update and delete todo List.


This is Multi-User Todo List WebApp using Django.
using this multi-user todoList app  User can Register and login himself.
after login user can  view his todo list also add/update/retrive/delete todo.
todo list show in the form of table :-
#Title:- show title of the Todo
#Status:- show todo is completed or pending.
#Priority:- show priority range(1-10)
#Action:-user can check(Pending/Completed) and Delete.
